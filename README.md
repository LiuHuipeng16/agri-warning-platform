# 农业病虫害智能预警辅助决策系统

## 项目简介

本项目面向智慧农业场景，构建集病虫害预警、农业知识管理、AI智能问答、图文分析与数据可视化于一体的农业病虫害智能预警辅助决策平台。

系统基于 Open-Meteo 气象数据、农业专家规则模型以及 RAG（Retrieval-Augmented Generation）知识增强生成技术，实现病虫害风险分析、智能预警、农业知识问答及 AI 辅助决策等功能。

* Spring AI

### 前端

* Vue3
* Element Plus
* ECharts

## 核心功能

### 病虫害智能预警

* TODAY实时预警
* FORECAST未来预警
* 自动规则匹配
* 风险评分分析

### AI农业助手

* 农业知识问答
* 悬浮AI助手
* 多轮上下文对话
* SSE流式输出

### AI图文分析

* 病虫害图像识别
* 环境风险
* 作物预警统计
* 季节趋势分析
* 高风险病虫害排行

## 项目亮点

* 基于风险评分模型实现病虫害风险量化评估
* 支持预警命中依据展示，提高结果可解释性
* 基于RAG架构实现农业知识增强问答
* 支持AI图像与环境协同分析
* 基于SSE实现AI分析结果流式返回
